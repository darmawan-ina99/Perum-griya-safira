# Aplikasi Warga Perum Griya Safira

Aplikasi ini adalah sistem informasi warga berbasis web untuk mengelola data warga, iuran, dan kegiatan di Perumahan Griya Safira.

## Fitur Utama
*   Manajemen Data Kepala Keluarga dan Anggota
*   Pencatatan Iuran Bulanan
*   Galeri Foto Kegiatan
*   Pengumuman Real-time
